# 🎨 Color Palette Picker

A simple and interactive Color Palette Picker built with **React**.

Users can select a color from a palette, preview it, view its hexadecimal code, and reset the selection.


## Live demo: https://color-palette-picker-lyart.vercel.app/

## 📸 Screenshot

![App Screenshot](./assets/screenshot.png)

## ✨ Features

- 🎨 Display multiple color buttons
- 🖱️ Select a color by clicking a button
- 👀 Live color preview
- 📝 Display selected color name
- 🔢 Display hexadecimal color code
- 🔄 Reset selected color
- 📱 Responsive layout

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/HayatAbdulfetah/color-palette-picker.git
```

### Navigate into the project

```bash
cd color-palette-picker
```

### Install dependencies

```bash
npm install
```

### Run the development server

```bash
npm run dev
```
